[S.B.L.]

.Reference



> Script Bot Language is a markup language for creating custom functions in the "SCRIPT" bot for Discord. It can be executed with the SCRIPT.code command or by programming it in tags.

Example code:
<msg Hello!> <delay 2000> <msg my name is SCRIPT>


General tags:

<msg>  <delay>  <choose>  <cmd>  <nick>  <name>  <id>  <ruser>  <rnumber>  <set>  <get>  <text>

SCRIPT.tag only:

<argument>  <*> 


#msg


The "msg" tag is used by SCRIPT to send a text message with the specified content.
WARNING: The message can mention @everyone or @here if the bot has permissions.
Example:
SCRIPT.code <msg Message1> <msg message2> <msg message3>


#delay


The "delay" tag is used to set a waiting time between instructions (in milliseconds).
Example:
SCRIPT.code <msg first message> <delay 5000> <msg second message (5 secs later)>


#choose


The "choose" tag is used to randomly select one value among several.
Example:
SCRIPT.code <msg the selected letter is: <choose A && B && C>>


#cmd


The "cmd" tag is used to execute a SCRIPT command (regardless of whether or not the user has permission to execute it).
Example:
SCRIPT.code <cmd say @everyone mention not allowed in say command.>


#nick


The "nick" tag is used to obtain a user's own nickname from their username or ID. If no argument is given, it returns the nickname of the person executing the code.
Example:
SCRIPT.code <msg Hello <nick>, hello <nick 653360060516270151>>


#name


The "name" tag is used to obtain a user's username from an ID. If no arguments are given, it returns the username of the person executing the code.
Example:
SCRIPT.code <msg Hello <name>, hello <name 653360060516270151>>


#id


The "id" tag is used to obtain a user's ID from their username. If no arguments are given, it returns the ID of the person executing the code.
Example:
SCRIPT.code <msg Hello John, your ID is: <id John>>


#ruser


The "ruser" tag is used to obtain the username of a random user on the server.
Example:
SCRIPT.code <msg Hello <ruser>>


#rnumber


The "rnumber" tag is used to obtain a random integer. If a single argument is entered, it will return a number from 0 to the specified number. If two values ​​are specified, it will return a random number within the range.
Example:
SCRIPT.code <msg 0 to 100: <rnumber 100>>
SCRIPT.code <msg 100 to 200: <rnumber 100 200>>


#set


The "set" tag is used to declare a temporary variable with a name and value of your choice.
Example:
SCRIPT.code <set USERNAME <ruser>>


#get


The "get" tag is used to obtain the value of a previously created temporary variable.
Example:
SCRIPT.code <set USERNAME <ruser>> <msg Hello <get USERNAME>!>

#text


The "text" tag is used to avoid executing code in SBL and instead work on it as plain text.
Example:
SCRIPT.code <msg <text <msg this tag will not be executed!>>>

SCRIPT.tag ONLY


WARNING: The following tags can ONLY be used in the SCRIPT.tag command.

SCRIPT.tag add matchMaker|<msg <ruser> x <ruser>>


#argument


The "argument" tag is used to get arguments entered in order when the code is triggered.
WARNING: By using this tag no one will be able to execute your code if you don't provide all the arguments.
Example:
SCRIPT.tag add matchMaker|<msg <argument> x <ruser>>
> matchMaker John


#<*>


The "<*>" tag is used to get all the remaining arguments as a text with spaces.
Example:
SCRIPT.tag add repeat|<msg <*>>
> repeat Hello world, its a say command with @everyone and @here.
